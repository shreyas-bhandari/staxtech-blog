# 📝 Blog Website (Full MERN Stack)

This is a complete blog website built using the **MERN stack** during my internship at **StaxTech**.

## 💡 Features
- ✍️ Create, Read, Update, Delete blog posts
- 🔐 Login Page UI
- 📄 Individual post pages
- 📦 Backend with MongoDB
- 📱 Responsive Design

## 🛠️ Tech Stack
- ⚛️ React (Frontend)
- 🟢 Node.js + Express (Backend)
- 🍃 MongoDB (Database)
- 🎯 REST API
- 💅 CSS
